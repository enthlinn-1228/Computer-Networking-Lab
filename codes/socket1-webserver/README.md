# 套接字编程作业1：Web服务器
作业用 nodejs 编写

#### 第一部分
开启服务器
```js
node server.js
```
打开浏览器访问 `localhost:8080/index.html` 或者 `localhost:8080/HelloWorld.html`

#### 第二部分
开启服务器
```js
node server2.js
```
再用 `client2.js` 访问服务器
```js
node client2.js localhost 8080 index.html
// or
// node client2.js localhost 8080 HelloWorld.html
```
